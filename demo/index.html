Through the following examples, we show detailed demos on how to use this solver. We showcase two common use cases by focusing on two example problems. For each problem, we answer the following questions:

  1. How to translate the problem mathematically to get \(A,b,c\) and \(\mathcal{C}_i\).

  2. How to formulate the training data.

  3. How to execute the solver.

  4. How to read the results and do inference.

The two specific examples are:

  • A Multi-Objective Optimization Problem
  • A Matching Problem
